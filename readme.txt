images文件夹存放图标
includes存放一些功能性函数
js存放脚本
lang存放语言切换的代码
pictures存相关项目的图片
Smarty是Smarty引擎代码，里面的templates放前端的模板
sql存放mysql类，里面的mysql_cls是今天才开始编写的sql类，里面的初始化链接部分针对不同的数据库可能需要修改（目前用的为Arthur机的mysql的db数据库）
config.php为Smarty配置PHP
index.php为首页PHP
lib.php是临时的测试功能性函数的PHP之一
testlib是临时的测试功能性函数的PHP之一
testmysql.php是临时的测试mysql类功能性的PHP
